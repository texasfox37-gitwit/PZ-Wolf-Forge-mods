⚠️ REQUIRED EXTERNAL JAVA BRIDGE

TDDUP / Till Death Do Us Part requires an external Java Bridge package from GitHub Releases.

Steam Workshop subscription alone is not enough.

Why?
Steam Workshop cannot include the required `.bat`, `.jar`, installer, or package files. This Workshop item contains the Workshop-safe Lua and media files only. The required Java Bridge is distributed separately through GitHub Releases.

Install steps:

1. Subscribe to this Workshop item.
2. Fully close Project Zomboid and Steam.
3. Download the required Java Bridge here:
   REPLACE_WITH_YOUR_GITHUB_RELEASE_URL
4. Extract the whole folder into your Project Zomboid main folder.
5. Open the extracted folder.
6. Run `installer.bat`.
7. Start TDDUP with `Launch_TDDUP_Bridge_Alternate.bat`.
8. Enable TDDUP in the in-game Mods menu.
9. Restart once more using `Launch_TDDUP_Bridge_Alternate.bat` before loading a save.

Important:
Normal Steam launch may open Project Zomboid, but it will not load the Java Bridge with v3.6.2. Steam Alternate Launch may also open the game without the Java Bridge. Use `Launch_TDDUP_Bridge_Alternate.bat` for TDDUP.

The installer copies the public TDDUP Java bridge jars into a `TDDUP_Bridges` folder inside your Project Zomboid folder. It also removes old managed TDDUP Java launch entries from `ProjectZomboid64.json` so normal launch is not left with the entries that caused hard-crashes for some users. It preserves other mods' existing non-TDDUP Java entries.

No telemetry. No credential collection. No background service. No save-file modification. No auto-downloads.

Only download the bridge from the official GitHub Releases page.
