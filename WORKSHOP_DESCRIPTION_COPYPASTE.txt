⚠️ REQUIRED EXTERNAL JAVA BRIDGE

TDDUP / Till Death Do Us Part requires an external Java Bridge package from GitHub Releases.

Steam Workshop subscription alone is not enough.

Why?
Steam Workshop cannot include the required `.bat` or `.jar` files. This Workshop item contains the Workshop-safe Lua and media files only. The required Java Bridge is distributed separately through GitHub Releases.

Install steps:

1. Subscribe to this Workshop item.
2. Fully close Project Zomboid and Steam.
3. Download the required Java Bridge here:
   REPLACE_WITH_YOUR_GITHUB_RELEASE_URL
4. Extract the zip into your Project Zomboid game folder.
5. Run `Launch_TDDUP_Bridge_Alternate.bat`.
6. Enable TDDUP in the in-game Mods menu.
7. Restart once more using `Launch_TDDUP_Bridge_Alternate.bat` before loading a save.

Important:
Normal Steam launch may open Project Zomboid, but it will not load the Java Bridge with v3.6.3. Steam Alternate Launch may also open the game without the Java Bridge. Use `Launch_TDDUP_Bridge_Alternate.bat` for TDDUP.

v3.6.3 has no installer and no uninstaller. The package contains the direct launcher and the two public TDDUP bridge jars. It does not edit `ProjectZomboid64.json`.

If an older TDDUP installer broke normal launch, use Steam's Verify integrity of game files option for Project Zomboid, or manually remove old TDDUP entries from `ProjectZomboid64.json`. Do not restore the newest TDDUP backup just because it is newest.

Advanced users who need TDDUP and another Java-agent mod to load through normal Steam launch can follow the manual `ProjectZomboid64.json` instructions on GitHub. Back up the file first. A bad JSON edit can stop the game from launching.

No telemetry. No credential collection. No background service. No save-file modification. No auto-downloads.

Only download the bridge from the official GitHub Releases page.
